# octosecret

[![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

# CLI
## Interactive
```shell
$ octosecret
```

## Encrypt
```shell
$ octosecret encrypt <username>
$ octosecret encrypt <username> [filename]
```

## Decrypt
```shell
$ octosecret decrypt
$ octosecret decrypt [filename]
```
